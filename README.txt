Проект по продвинутой работе с файлами, путями, кодировками и сериализацией данных.

Запускать поэтапно:
1. task1_filesystem.py
2. task2_processing.py
3. task3_backup.py
4. task4_schema.py
5. task5_report.py
или result.py — запускает все этапы подряд.

# Формат отчёта:
- Итоговый отчёт автоматически создаётся в output/final_report.json (JSON-отчёт).
- Весь процесс логируется в logs/setup.log (txt-лог).

# Комментарии
— Каждый скрипт содержит подробные комментарии к ключевым этапам и функциям.
— result.py снабжён поясняющими блоками для каждого этапа.
